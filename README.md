# nixos-config
## Introduction

This is my personal NixOS configuration made using Flakes and Home Manager. It is a
continuation of my [previous attempt](https://github.com/itsyunaya/nixos-config-old) of
getting aquainted with Nix, but now actually stable and usable as a daily driver

## Specifications

The configuration features a dual compositor setup with Niri and Hyprland, wherein it is
possible to switch between the two just by setting one config option. 

Further notable components include:
- Waybar, Anyrun app launcher and Hyprlock
- Kitty terminal with LaTeX-ready Neovim/Nixvim setup
- rmpc music player
- a few self-packaged applications

## Note

This config is still work in progress and may change drastically at any time
